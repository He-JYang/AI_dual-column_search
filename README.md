# English Edition
## Quick Start

**Clone or download the project, then double-click `ai_search_dual_panel.html` to experience simultaneous access to Bing and Kimi.**

To enable automatic questioning for Kimi, install the browser extension **Tampermonkey** and manually add the `Kimi_auto_ask.user.js` script from the project to Tampermonkey. You can create a new script directly and copy the file contents into it.

To query additional search engines or AI models simultaneously, configure all remaining scripts as needed.

To set `ai_search_dual_panel.html` as your browser homepage, install the extension **New Tab Changer**.

The `ScriptHistory` folder contains legacy scripts for development reference only—no installation or usage is required for this folder.


# 中文版
## 快速开始

**克隆或下载项目后，直接双击 `ai_search_dual_panel.html` 打开即可体验 Bing 和 Kimi 同时访问。**

如果需要让 **Kimi** 能够自动提问，需要安装浏览器插件 **篡改猴（油猴 / Tampermonkey）**，并手动将项目中的 `Kimi_auto_ask.user.js` 添加为油猴脚本。可以直接手动点击创建脚本，然后将项目中的文件内容复制过去。

如需同时对更多搜索引擎或 AI 模型提问，需要将剩余脚本全部配置。

如果需要将 `ai_search_dual_panel.html` 设置为浏览器主页，需要安装插件 **New Tab Changer**。

`ScriptHistory` 为历史脚本，作为开发参考，仅安装无需使用该文件夹。
